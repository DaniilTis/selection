Виды селекторов

Здесь h3 имя произвольного элемента. Следует понимать, что хотя правила стиля можно применить к любому элементу, результат будет заметен только для элементов, которые непосредственно отображаются в контейнере body.

  • Классы применяют, когда необходимо определить стиль для одного или нескольких элементов веб-страницы.
  • При описании идентификатора вначале указывается символ решётки (#), затем идёт имя идентификатора. ID должен быть уникальным для страницы
  • CSS селектор потомков или CSS селектор дочерних элементов Его используют если необходимо выбрать элементы определенного типа из множества дочерних элементов. Например, нужно выделить все ссылки, которые находятся в элементе li.
